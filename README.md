## 🗂️ 스터디 규칙

### 주제 선정 방식

1. 매주 학습할 대주제를 선정한다. (Java, Spring, OS, Network, DB 중 택1)
2. 각자 학습할 소주제를 제시한다.
3. 사다리타기로 학습할 소주제를 확정한다.

### 스터디 진행 방식

1. 매주 월요일 주제를 선정한다.
2. 대주제 이슈를 생성한다.
   - 예시: `[1주차] Java`
3. 각 소주제를 대주제 이슈의 sub-issue로 생성한다.
   - 예시: `[1주차] Java - 클래스`
4. 일주일 동안 해당 소주제에 대해 각자 학습한 뒤 Markdown 형식으로 정리한다.
   - 예시: `Java/01_클래스.md`
5. 스터디 진행 당일 각자 맡은 부분에 대해 5-10분 간 발표를 진행한다.
